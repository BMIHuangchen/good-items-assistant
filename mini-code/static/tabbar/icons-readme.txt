TabBar 图标说明
================

当前小程序使用文字 TabBar，不强制依赖本地图标文件。

如果后续需要添加图标，建议准备 8 个 PNG 文件：
1. home.png / home-active.png - 首页图标
2. category.png / category-active.png - 分类图标
3. favorites.png / favorites-active.png - 收藏图标
4. me.png / me-active.png - 我的图标

图标规格建议：
- 尺寸：81px x 81px
- 格式：PNG
- 文件大小：不超过 40KB
- 普通状态：#7A847E
- 选中状态：#2F6B4F

注意：
- 本项目不包含购物车、订单、支付等经营性能力。
- 不要使用 cart、order、payment 等旧电商命名。
